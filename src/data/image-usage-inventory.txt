================================================================================
                           IMAGE OPTIMIZATION NOTES
================================================================================

1. RESPONSIVE IMAGES: Many banners include multiple size variants (640, 960,
   1280, 1920) for optimal loading across devices.

2. FORMAT PREFERENCES:
   - WebP: Primary format for photos and illustrations (better compression)
   - AVIF: Modern format for hero banners (best compression)
   - SVG: Logos and icons (vector, scalable)
   - PNG: Logos with transparency, screenshots

3. LAZY LOADING: Property images should use lazy loading for better
   performance.

4. ALT TEXT: All images should have descriptive alt text for accessibility.
